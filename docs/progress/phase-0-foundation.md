- Chức năng hoàn thành: backend FastAPI, frontend Next.js, mixin cơ sở, soft delete filter, migration đầu tiên.
- Kiến thức/công nghệ đã dùng: SQLAlchemy 2.0, Alembic, Pydantic v2, TestClient, Next.js App Router.
- Rủi ro/nợ kỹ thuật: chưa có realtime, chưa có upload file thật.
- Demo: truy cập `GET /`, `GET /api/v1/health`, trang chủ frontend.
- Kế hoạch phase tiếp theo: đào sâu user/profile/auth và polish session UX.
